data/number-values.json
~~~~~~~~~~~~~~~~~~~~~~~

The file contains a JSON object whose names are characters and values
are their descriptions, which are JSON objects with following
name/value pair:

  cjk_numeral [number]

    The numeral value when the character is to be interpreted as a CJK
    number, per the Numbers specification
    <https://manakai.github.io/spec-numbers/>.

* Source

Numbers <https://manakai.github.io/spec-numbers/>.
